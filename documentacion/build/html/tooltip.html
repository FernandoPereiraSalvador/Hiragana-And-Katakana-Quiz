tooltip module¶

class tooltip.Tooltip[fuente]¶

Bases: object

Clase que maneja la creación y visualización de tooltips (información emergente) en widgets.

Attributes: - tooltip: Ventana emergente que muestra el tooltip.

Métodos: - __init__(self): Inicializa una nueva instancia de Tooltip. - mostrar_tooltip(self, widget, text_dict): Muestra un tooltip en un widget específico. - ocultar_tooltip(self): Oculta el tooltip si está visible.

mostrar_tooltip(widget, text_dict)[fuente]¶

Muestra un tooltip en un widget específico con información proporcionada en text_dict.

Parámetros: :param widget: El widget en el que se mostrará el tooltip. :param text_dict: Un diccionario que contiene la información a mostrar en el tooltip.

Detalles: La función calcula las coordenadas en las que se debe mostrar el tooltip cerca del widget. Crea una nueva ventana emergente (Toplevel) y configura su apariencia y contenido utilizando el contenido de text_dict. El tooltip se ajusta automáticamente en tamaño para acomodar su contenido.

Devuelve:

None

ocultar_tooltip()[fuente]¶

Oculta el tooltip si está visible.

Detalles: La función verifica si hay un tooltip mostrándose en la ventana actual. Si es así, se destruye la ventana del tooltip, ocultándolo.

Devuelve:

None
